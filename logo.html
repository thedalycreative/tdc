Images & Logo Guidelines

Logo Usage

  • Use LOGO-01.png (880x740px) as the main logo for all digital, print, and overlays. Always use the transparent PNG for flexibility on any background.
  • For favicon and small icons, use FAVICON.png (272x250px) or FAVICON-00.png (399x367px).
    Keep at least 32x32px visible for browser tabs.
  • Never stretch, skew, add effects, or recolour the logo. Leave at least 16px padding around all sides.
  • Use the file path ./tdcreative-images/LOGO-01.png for HTML (and same structure for all images below).
  • Example usage: <img src="./tdcreative-images/LOGO-01.png" alt="The Daly Creative logo">

Swirl/Gradient Graphics

  • For backgrounds, overlays, and brand accents, use SWIRL-01.png (1920x1080px) as your main swirl file for full-screen width.
    Use SWIRL-02, SWIRL-03, etc. (1359x1088px or 1920x1080px) for alternate swirl graphics or cropped section highlights.
  • Keep original colour gradients and aspect ratios.
  • For backgrounds: background-image: url('./tdcreative-images/SWIRL-01.png');
  • For design flexibility: SQUIRL-00, SQUIRL-01, etc. are also available in 1359x1088px.

Image Naming & Folder Structure

  • All images live in: ./tdcreative-images/
  • Filenames must follow: ImageType-##.ext (e.g. LOGO-01.png, HEADER-01.png, SWIRL-04.png)
  • When adding new images, increment the number and use a consistent descriptive prefix.
  • Always keep image alt text clear and specific for SEO/accessibility.
